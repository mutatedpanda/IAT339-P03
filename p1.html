Project 1

Description

This project was the final project for my Mobile Computing course (IAT 359). Students got the freedom to create any application on the phone that met a set of requirements.

Details

Type: Individual Project

Programs Used: Android Studio, Adobe Illustrator

Requirements: have at least three activities, explicit and implicit intents, data storage in local SQLite Database and Shared Preferences (the ability to add/remove data and perform queries in the database), and device hardware usage (sensors and GPS or camera functionality

About

The application that I chose to make was a gaming application called Fallen. This game was inspired by DoodleJump. The players would play as a character that was falling and would have to dodge obstacles along the way down. It was an endless game, so players can compete with their friends to see who could get a higher score.

Difficulties & Resolutions

The difficulties I encountered while coding the game were coding game functionalities in Android Studio without using a third-party program. To resolve this, it required a lot of researching, which resulted in finding a method to do so.
